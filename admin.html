Deploy the Google Apps Script proxy

One-time setup — fixes the CORS error that prevents direct Sheet access from a browser

Browsers block direct Google Sheets requests (CORS). The fix is a tiny Google Apps Script Web App that lives inside your own workbook and serves the data without any restrictions.

  1. 1 Open your Google Sheets workbook → click Extensions → Apps Script.
  2. 2 Delete all existing code. Paste the script below, then click Save (Ctrl+S).
  3. 3 Click Deploy → New deployment. Type: Web app. Execute as: Me. Who has access: Anyone. Click Deploy and copy the Web App URL.
  4. 4 Paste that URL into the "Apps Script URL" field in Step 2 below. 
function doGet(e) {
  var sheetName = e.parameter.sheet;
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var sheet = ss.getSheetByName(sheetName);
  if (!sheet) {
    return ContentService
      .createTextOutput(JSON.stringify({ error: 'Sheet not found: ' + sheetName }))
      .setMimeType(ContentService.MimeType.JSON);
  }
  var data = sheet.getDataRange().getValues();
  var output = JSON.stringify(data);
  return ContentService
    .createTextOutput(output)
    .setMimeType(ContentService.MimeType.JSON);
}
